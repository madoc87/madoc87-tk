Aumente sua Produtividade com IA Gratuita no VSCode (Llama 3 Copilot)

Introdução à Llama 3 e integração com o VS Code

Imagine poder integrar o Llama 3 à sua aplicação VS Code. Agora isso é possível! Baixe o Llama 3 em seu computador usando o npm e crie um co-piloto privado com IA. Sem isso, você precisa escrever código manualmente, corrigi-lo manualmente e refatorá-lo manualmente, o que resulta em menos produtividade e qualidade de código reduzida. Mas como evitar isso?

Em vez de buscar online por código, como conseguir todo o código diretamente no VS Code? Com isso, você pode obter explicações para seu código, refatorar o código, criar documentos para o código selecionado e corrigir bugs.

Baixando e configurando o Llama 3

Estou animado para mostrar como usar o Llama 3 com o VS Code usando IA como seu companheiro de código. Com ele, você pode criar rapidamente qualquer código básico, corrigir e refatorar código na hora, aumentando assim a produtividade, a qualidade do código e reduzindo erros.

Antes de começarmos, lembre-se de que eu regularmente crio vídeos sobre Inteligência Artificial em meu canal do YouTube. Inscreva-se e ative o sino para ficar atualizado. Clique no botão de curtir para ajudar outros a encontrarem este vídeo.

Se você ainda não tem o VS Code, baixe-o no site do Visual Studio Code. Em seguida, baixe o Llama no site do ama.com. Abra o VS Code e clique no ícone de extensões. Procure por "code GPT", escolha a opção com 1 milhão de downloads e instale-a. Clique no ícone novamente, vá para as configurações da extensão e escolha o provedor Llama no menu suspenso. Em seguida, selecione o modelo Llama 38b. Estamos prontos para codificar!

Escrevendo e executando seu primeiro script de IA

Agora, vamos criar um arquivo chamado "app.py". Clique no ícone de arquivo e crie um novo arquivo. Abra o editor de código e clique no ícone do code GPT. Selecione o provedor como Llama e o modelo como Llama 38b.

Como ponto de partida, vou pedir para escrever um código de API do Flask. Ele gera a resposta rapidamente. Agora, posso estender esse código ainda mais. Por exemplo, conectando-o a um banco de dados SQLite. Posso pedir ao Llama para fazer isso por mim.

Depois, posso pedir para corrigir erros. O Llama identifica facilmente onde está o problema e fornece uma solução rápida. Além disso, posso refatorar o código e adicionar documentação com apenas alguns cliques.

Em apenas 5 minutos, criamos um código que interage com um banco de dados e responde às consultas dos usuários. Estou empolgado com essas possibilidades!
